Opinión

Hecho por: Juan Felipe Jimenez Pacheco.

En este artículo se pretende mejorar la cultura del cine y explorar más la ley del desarrollo de la red social, lo cual me parece una buena idea para un proyecto de investigación, pero sentí que podían complementar y utilizar otros medios para desarrollar este proyecto como encuestas, o preguntar a personas que no tengan acceso a internet o algún medio tecnológico.

Me agrado que describieran los comportamientos que las personas tenían en la opinión de películas. Por qué que esto es muy útil para hacer análisis y minería de datos para obtener resultados dependiendo de las actitudes de la gente. En la parte de diseño me aprecio de que trabajaran con MVC, ya que este es una arquitectura de software muy útil y fácil de entender, pero sobre todo fue buena idea que implementaran el modo de arquitectura B/S haciendo que el sistema de jerarquía sea más claro y pueda mejorar aún más la expansibilidad del sistema.

Vue.js y Node.js son las dos principales tecnologías de desarrollo de este sistema. En mi opinión nunca he usado estos Frameworks he escuchado que son de gran ayuda para agilizar código. En este proyecto se desarrolla la página web de forma por componentes con Vue.js, que reduce el ciclo de desarrollo hasta cierto punto.

Este proyecto se aplica el marco Express de Node.js para diseñar la API de enrutamiento y codificar programas de lógica empresarial. Y por último se selecciona MongoDB como la base de datos del sistema. Todas las tecnologías que se usan en este proyecto son muy nuevas, eso fue lo que más me encanto porque ayuda a conocer que tecnologías se están utilizando en mundo laboral y así ayuda a no quedarse en las tecnologías, además de que el soporte que tendrá este sitio web será de largo tiempo usando estas herramientas que agilizan y mejoran los tiempos de espera al cliente.
